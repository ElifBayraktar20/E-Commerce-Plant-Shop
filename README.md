# 🌿 GREENMIND – E‑Commerce Plant Shop

Bu proje, doğa ve bitki severler için hazırlanmış modern bir bitki e‑ticaret (online mağaza) UI/UX front‑end tasarımıdır.
GREENMIND, HTML & CSS kullanılarak geliştirilmiştir. Proje, kullanıcı dostu bir tasarım ve temiz kod yapısı ile responsive (duyarlı) bir frontend deneyimi sunar.
